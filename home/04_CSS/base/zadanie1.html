Форматы файлов.

.Doc или .doc(от англ. document ) -расшиоение имени файла, используемок для файлов
представляющих текст, с разметкой или без. Расширение .DOC часто использовалось для
обозначения простых текстовых файлов без форматирования, однако позже стало
использовать для двоичных форматов с разметкой.

Документ MS Word

Portable Document Format(PDF) -межплатформенный формат электронных документов,
разработанный фирмой Adobe Systems с использованием ряда аозможностей языка
Postscript.

Документ PDF

ZIP - популяный формат сжатия данных и архивации файлов.Файл в этом формате
обычно имеет расширение .zip и хранит в сжатом или несжатом виде один или несколько
файлов, которое можно из него извлечь путем распаковки с помощью специальной
программы.
